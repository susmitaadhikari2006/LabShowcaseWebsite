What is BPA?


BPA is a national organization which the Programming and Software Development lab is part of. BPA hosts conferences and competitions where we take the skills we learn in lab and apply it in competitions! Some of the competitions include C#, Java, web-development using HTML and CSS, game development, Excel, and web-development Team. You may also participate in other contests including speech, and presentations.
