In this is lab you will be treated Professionally and you will work in you own little cubical. You will work on several hands on project and will also be able to collaborate with your peers.

Juniors start with c# and create aplications and softwares with it! They then move on to and have the chance to learn unity. For the end of the semester project they may choose to creating a software or game of their choice depening on the perameters of the project. They then move on to web development using HTML, CSS, and JavaScript!

Seniors start with JAVA and create aplications and softwares with it! They then move on to mobile application development using react and reactJS!
